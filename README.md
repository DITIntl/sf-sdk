# 顺丰 Python SDK

官方SDK只是一个demo，而且数据格式为XML，处理起来并不方便，因此起了这个项目并将数据格式调整为了json格式。

## 使用说明

### 安装

```python
pip install sf-sdk
```
使用
```python
from sf.api import SF

sf = SF("clientcode","checkword")

sf.order.create_order(clientid,..)

```
